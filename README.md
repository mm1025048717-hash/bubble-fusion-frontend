# Bubble Fusion Lab – AI 驱动的创意融合 🚀

**Bubble Fusion Lab** 是一种全新的**思考、整理和创造**方式。我们将您的想法转化为可视化的**气泡** —— 当您将两个气泡拖拽到一起时，在 **AI** 的驱动下，它们会**融合**成全新的内容。

---

## ✨ 为什么选择 Bubble Fusion？

*   **视觉优先** – 每个想法都是一个气泡。整理、拖拽并连接它们。
*   **AI 驱动融合** – 将两个气泡放到一起，AI 会即时生成新的标题和洞察。
*   **多格式支持** – 管理文本、图片、视频、文档和音频。
*   **随处导出** – 导出为 JSON 或 Markdown 格式的结构化输出。
*   **轻量级演示** – 立即尝试，无需注册。
